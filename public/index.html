About

The UWSO to Table Converter is a straightforward application that converts data from the Surface Bbservations section of Wyoming Weather Web into a CSV file, making subsequent data analysis easier.

How to use it
  1. Open Global Surface Observations on Wyoming Weather Web
  2. Select the region
  3. Choose the station and generate data
  4. Copy the data and paste it into the text input
  5. Click the 'Convert to Table' button
  6. Download the generated CSV file
